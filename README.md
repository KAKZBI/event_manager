📅 Event Manager: Reviving an Archived Classic
This project is a bit of a "resurrection." I found it in the archived section of The Odin Project curriculum. Even though it's no longer part of the main path, I really liked the idea of working with real-world data and APIs, so I decided to tackle it anyway.

You can find the original project instructions here https://github.com/TheOdinProject/curriculum/blob/809f1e440e63fb56cf1b86fcc8e8be777aaf730d/archive/ruby/files_and_serialization/project_event_manager.md.

🧩 The Challenge: Fixing a Broken Tutorial
The biggest hurdle was that the original tutorial relies on the Google Civic Information API, which is no longer available or easy to access for this type of learning.

Instead of giving up, I decided to pivot. I integrated the Geocodio API to handle the legislator lookups. It was a great lesson in "real-world" programming: sometimes the tools you're told to use break, and you have to find a new way to get the job done.
